# _.modulo(divisor, dividend)

Determimes the modulus of the `dividend` `modulo` `divisor` (e.g., `1 % 2 = 1`).

## Arguments

* `divisor (Number)`: The divisor number.
* `dividend (Number)`: The dividend number.

## Returns

* `(Number)`: The modulus.

## Examples

### Equivalence

```javascript
_.modulo(2, 1) === 1 % 2
// => true
```
