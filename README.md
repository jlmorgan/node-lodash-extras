# Lodash Extras

Extensions for [lodash][] primary targeting [Node.js][] applications.

## Installation

Using npm:

```bash
npm install --save @jlmorgan/lodash-extras
```

## Methods

* `append`
* `coalesce` (alias: `alt`)
* `isBlank`
* `isJust`
* `isNothing`
* `prepend`
* `setProperty`
* `setValue`

[lodash]: https://lodash.com/
[Node.js]: https://nodejs.org/
